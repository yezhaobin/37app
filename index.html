37app render app can receive dlna media controller (like bubbleupnp ,imediashare or order dmc app) command and stream. You can use dlna media controller to control 37app render playback. Software customization or buy a license，please contact me，my email：yezhaobin@gmail.com.

youku 视频地址:http://v.youku.com/v_show/id_XNDc4MDE0Mjc2.html

android2.2 或以上系统版本的手机 、平板 、minipc 、android box，装上37app dlna render 这个程序之后，服务运行在系统的后台，系统可以继续运行其他任意程序。另一个手机上的控制软件（视频中手机用到的bubbleupnp作为控制端，其它类似的软件还有如三星手机的娱乐共享软件，或者市场上的其他支持dlna 播放控制的软件，如android版本的腾讯视频）可以轻易的把视频，音乐，图片推送到安装了37app dlna render 软件的android 机器上播放，render 实现暂停，退出，切换，进度拖动等常规的播放功能。视频为简单的演示demo。欢迎交流咨询，联系详见联系页面。
